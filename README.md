# ToseeWritest
该项目是参与2020年高校小程序大赛的"笔试一下"微信小程序的服务端程序。
## 技术选型
* MySQL 5.7.25
* Maven 3.3.9
* Java 1.8.0_201
* SpringBoot 2.2.6

## 运维环境
* centos 7
* nginx
* vsftpd
* Java 1.8.0_201

## 注意事项
值得注意的是，该项目用了`lombok`插件，如果Idea未安装该插件，会导致"方法找不到"错误。
